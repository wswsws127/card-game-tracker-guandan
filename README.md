# Card-register
掼蛋手动记牌器

https://wswsws127.github.io/card-game-tracker-guandan/

**使用方法：**
用加减箭头(±1)/炸弹按钮(-4)/键盘输入 来减少场上现存牌的库存。按reset重置。建议游戏开局时快速输入自己手牌。

键盘输入对应（大小写随意）：
D - 大王 
X - 小王
A 或 1 - A
KQJ - KQJ
S - 10
9到2 - 9-2
